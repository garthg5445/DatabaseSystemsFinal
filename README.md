# DatabaseSystemsFinal
# Boardwalk Casino Database Project

**Course:** Database Systems, Fall 2025

---

## Overview
This project involved designing and implementing a fully normalized relational database for a boardwalk casino arcade. The database models the interactions between customers, games, transactions, prizes, and ticket redemptions, ensuring data integrity and consistency throughout.

---

## Key Features
- **Entity Modeling:** Designed comprehensive ER diagrams for all entities including customers, games, transactions, prizes, and ticket redemptions.
- **Relational Database Implementation:** Created a fully normalized schema in SQL.
- **Data Integrity:** Implemented primary and foreign key constraints to enforce referential consistency.
- **SQL Operations:** Developed queries to insert, update, and manage structured data within the database.
- **Transactional Support:** Ensured accurate tracking of game plays, ticket earnings, and prize redemptions.

---

## Technologies Used
- **SQL** – schema design, queries, and data manipulation
- **ER Diagrams** – for entity modeling and normalization
